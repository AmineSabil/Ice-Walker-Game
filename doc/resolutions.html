resolutions module¶

Ce module traite la partie concernant la resolution du jeu.

Fonctions¶

resolutions.moves(grille, tup_config)[source]¶

Cette fonction permet de générer toutes les configurations possibles des joueurs en faisant tous les mouvements possibles

Parameters:
  • grille (Grid) – la grille du jeu
  • tup_config (tuple) – une configuration données des joueurs
Returns:

une liste de toutes les configurations des joueurs avec tous les mouvements possibles
Return type:

list
CU:

None
resolutions.is_solved(grille, tup_config)[source]¶
Cette fonction indique est-ce-que le joueur principal est bien arrivé à la case finale dans grille
Parameters:
  • grille (Grid) – la grille du jeu
  • tup_config (tuple) – une configuration données des joueurs
Returns:

True si le joueur principal est bien arrivé à la case finale ,False sinon
Return type:

bool
CU:

None
Exemples: 
>>> g=Grid.from_file('grid02.txt')
>>> is_solved(g,(((2,7),),()))
True
>>> is_solved(g,(((1,3),),()))
False
resolutions.fonction_resolution(grille)[source]¶
Cette fonction permet de savoir tous les coordonnées par lesquelles le joueur principal va passer pour arriver à la case finale.
Parameters:grille (Grid) – la grille du jeu
CU:None
Exemple: 
>>> g=Grid.from_file('grid06.txt')
>>> fonction_resolution(g)
